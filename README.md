# PROTAC Unsupervised ML Discovery

## Project Overview
This repository contains the implementation of unsupervised machine learning approaches applied to PROTAC molecules for next-generation anti-cancer drug discovery using targeted protein degradation (TPD).

## Objectives
- Molecular fingerprint generation
- Butina clustering
- Physicochemical property analysis
- Chemical space visualization (PCA/t-SNE/UMAP)
- Scaffold analysis

## Methods
- RDKit
- Unsupervised clustering (Butina algorithm)
- Molecular similarity (Tanimoto coefficient)
- Descriptor calculation (HBD, HBA, MW, LogP, TPSA, RB)

## Applications
Identification of structurally distinct PROTAC clusters for targeted protein degradation in oncology.
